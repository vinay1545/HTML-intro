HTML

(Hyper Text Markup Language)

HTML (HyperText Markup Language) is the standard language used to create and design webpages. It structures content using tags like headings, paragraphs, links, and images. HTML defines the elements on a webpage, providing a foundation for creating and displaying content on the internet in a browser.

Structure of HTML:

The structure of an html document includes:
1.Doctype html:Declares the document type and version of HTML.
2.Root element:The root element that contains the entire webpage content.
3.Head:Contains metadata, links to external files (like CSS), and the page title.
4.Body:Contains the visible content of the webpage, like text, images, links, and more.

Types of HTML Tags:

There are two types of html tags:
1.Paired or Container tags:Paired tags (also known as container tags) consist of two tags an opening tag and a closing tag.
Ex:h1-h6,p,b,head,title.......
2.Unpaired or selfclosing tags:An unpaired tag (also known as a self-closing tag or empty tag) is a tag that does not have a closing counterpart.
Ex:br,img,hr,input.....

Elements of HTML:

The HTML element is everything from the start tag to the end tag.
There are three types of elements in html, there are as follows:
1. Block Level Elements:The tag which occupies whole width and creates a new line.
Ex:h1-h6,, div,semantic elements...
2.Inline-Block Element:The elements which occupy only require space and does not create a new line and we can apply width,height,margin andpadding properties.
Ex:img, iframe,input, button...
3. Inline Element:The elements which occupies only required space and does not create a new line,in css we cannot apply width, height,padding top and bottom and margin top and bottom.
Ex:span, formatting tags, b, strong...
